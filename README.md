# Accordion-Read-More-Toggle
Toggle hidden content sections with smooth transitions. Button text updates automatically between “Read More” and “Read Less.” Perfect for FAQs, content previews, or collapsible UI sections.

## Usage
1. Add a button with class `js-toggle-button`.
2. Add `data-target="#id-of-hidden-content"` to the button.
3. Hidden content should have class `hidden` for initial state.
